AI Engineer Learning Toolkit - Test Suite

Open each test file in a browser to run its unit tests.

  • test_bilingual.html -- I18n module, Chinese translations, content structure, language toggle
  • test_highlights.html -- HighlightManager: apply, remove, colors, localStorage persistence
  • test_search.html -- Search module: query matching, case insensitivity, special chars, result limits
  • test_comments.html -- CommentModule: save, retrieve, remove, modal, tooltip
  • test_chat.html -- ChatModule: context banner, API key, localStorage keys, UI elements
All tests are self-contained HTML files. No npm or build tools required.
